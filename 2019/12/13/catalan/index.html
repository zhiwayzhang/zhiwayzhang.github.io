小兔的棋盘 组合数学

2019-12-13 ACM ACM, 数论 Comments Word Count: 522(words) Read Count: 2(minutes)

Description

小兔的叔叔从外面旅游回来给她带来了一个礼物，小兔高兴地跑回自己的房间，拆开一看是一个棋盘，小兔有所失望。不过没过几天发现了棋盘的好玩之处。从起点(0，0)(0，0)(0，0)走到终点(n,n)(n,n)(n,n)的最短路径数是C2nnC_{2n}^{n}C2nn​,现在小兔又想如果不穿越对角线(但可接触对角线上的格点)，这样的路径数有多少?小兔想了很长时间都没想出来，现在想请你帮助小兔解决这个问题，对于你来说应该不难吧! n≤35n \leq 35n≤35

Solution

由于规定不能超过对角线，可用分治的思想，只考虑沿对角线分隔开的三角形的情况，对于一个三角形中，求从(0，0)(0，0)(0，0)走到终点(n,n)(n,n)(n,n)的最短路径，观察发现无论怎么走，设到达某一点时向上走了iii步，向右走了jjj步，都有i≤ji \leq ji≤j这也能通过线性规划相关知识证明。不难发现只是一个类括号匹配问题，可用CatalanCatalanCatalan数求解，计算C(n)C(n)C(n)后即是在一个三角形中的解，答案是2C(n)2C(n)2C(n)

C(n)=∑i=0n−1C(i)⋅C(n−i−1)C(n) = \sum_{i=0}^{n-1} C(i) \cdot C(n-i-1)C(n)=∑i=0n−1​C(i)⋅C(n−i−1)
通项公式C(n)=C2nnn+1C(n)=\frac{C_{2n}^{n}}{n+1}C(n)=n+1C2nn​​

预处理C(n)C(n)C(n)即可

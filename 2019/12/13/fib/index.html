Triangle Fibonacci+二分查找

2019-12-13 ACM ACM, Math Comments Word Count: 597(words) Read Count: 3(minutes)

Description

有nnn个木棍，长度为1,2,3...n1,2,3...n1,2,3...n，现在要从中删掉一些木棍，使得剩下的木棍不能构成三角形，使删掉木棍的数量最少。T组数组，T≤20T \leq 20T≤20
n≤20n \leq 20n≤20

Solution

由于数据范围很小，可以直接暴力求解，依次选取两个数a,b(a<b)a,b(a<b)a,b(a<b)相加，要知道不能有任何一个数小于这个值，直接删掉(a,a+b)(a,a+b)(a,a+b)范围中的数即可
如果nnn的范围是1e91e91e9呢？
通过找规律发现我们剩下的数是这样的
1,2,3,5,8,13,21...1,2,3,5,8,13,21...1,2,3,5,8,13,21...
这是Fibonacci数列!!!
所以我们只需要找到≤n\leq n≤n的Fibonacci数有几个，减去就是答案
可以直接lower_bound注意处理极限数据！！！即n==1∣∣n==2n==1||n==2n==1∣∣n==2的情况
也可手写二分，但是二分貌似常数有点大，又考虑到第888888个Fibonacci数就爆掉1e91e91e9了，所以直接便利也完全没问题

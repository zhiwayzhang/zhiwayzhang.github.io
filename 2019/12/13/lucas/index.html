DP? 素数筛+Lucas定理+费马小定理

2019-12-13 数论►ACM ACM, Lucas定理, Math, 费马小定理 Comments Word Count: 1k(words) Read Count: 5(minutes)

Description

在杨辉三角中，从第一行第一列(0,0)(0,0)(0,0)开始，每次可选择向正下方走或向右下方走，走到第nnn行时不能超过第nnn行第kkk个元素，询问所经过路径的值的和的最小值mod(p)mod(p)mod(p)，一共有T组询问，T≤100000T \leq 100000T≤100000， 0≤k≤n≤1e90 \leq k \leq n \leq 1e90≤k≤n≤1e9, 保证ppp是质数， 其中p<1e4p < 1e4p<1e4

Solution

名字虽然是DP，但是可以找出最优方案。
优先考虑k≤n2k \leq \frac{n}{2}k≤2n​的情况，先向下走n−kn-kn−k步到达(n−k−1,0)(n-k-1,0)(n−k−1,0)
再一路沿着右下方走，直到到达底部，即有Cn−k0+Cn−k+11+Cn−k+22+...+Cnk=Cn+1kC_{n-k}^{0}+C_{n-k+1}^{1}+C_{n-k+2}^{2}+...+C_{n}^{k}=C_{n+1}^{k}Cn−k0​+Cn−k+11​+Cn−k+22​+...+Cnk​=Cn+1k​
通过变换Cn−k0=Cn−k+10C_{n-k}^{0} = C_{n-k+1}{0}Cn−k0​=Cn−k+1​0
再通过公式Cnm=Cn−1m+Cn−1m−1C_{n}^{m}=C_{n-1}^{m}+C_{n-1}^{m-1}Cnm​=Cn−1m​+Cn−1m−1​可将上述公式合并
得到Cn+1kC_{n+1}^{k}Cn+1k​
所以答案为Cn+1k+n−kC_{n+1}^{k}+n-kCn+1k​+n−k
当k>n2k > \frac{n}{2}k>2n​时，根据对称性，令k=n−kk=n-kk=n−k即可转化成上一种情况
又发现题目涉及组合数取模，所以要用到费马小定理

费马小定理:
假如aaa是一个整数，ppp是一个质数，且gcd(a,p)=1gcd(a,p)=1gcd(a,p)=1，即a,pa,pa,p互质，那么有ap−1≡1(modp)a^{p−1}≡1(modp)ap−1≡1(modp)

已知ap−1≡1a^{p-1}≡1ap−1≡1，可以得到a⋅ap−2≡1a \cdot a^{p-2}≡1a⋅ap−2≡1，我们称aaa和ap−2a^{p-2}ap−2为在mod(p)mod(p)mod(p)意义下的乘法逆元
然而这只解决了除法取模的问题，注意到nnn的范围在1e91e91e9直接计算组合数又是铁套老鹅(TLE)，于是借助Lucas定理

对于质数ppp,有Cnm mod p=C⌊np⌋⌊mp⌋⋅Cn mod pm mod p mod pC_n^m\ mod \ p = C_{\lfloor \frac{n}{p} \rfloor}^{\lfloor \frac{m}{p} \rfloor} \cdot C_{n\ mod\ p}^{m\ mod\ p} \ mod \ pCnm​ mod p=C⌊pn​⌋⌊pm​⌋​⋅Cn mod pm mod p​ mod p

可知n mod pn\ mod\ pn mod p和m mod pm\ mod\ pm mod p一定是小于ppp的数，可直接求解，其余部分继续用Lucas定理求解，当m=0m=0m=0的时候返回111

有了这些工具之后就可以预处理阶乘和逆元了

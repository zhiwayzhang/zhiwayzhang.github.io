0x02 存储历史与发展

硬件上的发展

1956–HDD

1984–Flash Memory 早期用于嵌入式

2010s–Persistent Memory 3D-Point Intel

SSD NAND Flash：

SLC，MLC，TLC，QLC存储单元中电平的等级

对比HDD和SSD：

  • 对于NAND，需要先擦出，再写

  • NAND Flash有擦除，HDD只能覆盖写

  • SSD多个颗粒可以并行I/O，总体上速度快

软件上的发展

网络存储 远程挂载目录NAS

分布式文件系统GFS，Hadoop HDFS

键值存储Key-Value

云存储 数据存在远端，商业化，软件协作Office
